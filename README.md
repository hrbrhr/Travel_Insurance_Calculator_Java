# Travel_Insurance_Calculator_Java
Этот проект представляет собой веб-приложение "Travel Policy Calculator" и призван помочь крупному банку изучить спрос на новую услугу — 
страхование путешествий, предложив клиентам инструмент для расчёта стоимости страхового полиса.

Разработка приложения даст возможность провести рыночное исследование и принять обоснованное решение о дальнейшем развитии данного направления.
